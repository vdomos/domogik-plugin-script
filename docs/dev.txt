xPL messages
============

xpl-cmnd
--------

The **exec.basic** message is used: ::

    xpl-cmnd
    {
    ...
    }
    exec.basic 
    { 
		pid: 		Command type, must be set to "cmd_action|cmd_info"
		program: 	Executable filename, including path and extension
		arg:		Command line arguments or 'none'
		status: 	'start' for running program (for "Script Action" or "Script Sensor" device)
    }



xpl-trig
--------

The answer of the xpl-cmnd.

The **exec.basic** message is used: ::

    xpl-stat
    {
    ...
    }
    exec.basic 
    { 
		pid: 		Command type, must be set to "cmd_action|cmd_info"
		program: 	Executable filename, including path and extension
		arg:		Command line arguments or 'none'
		status: 	"executed|value|failed"  "executed" for a "Script Action" device , value for a "Script Sensor" device, "failed" for all failed script.
    }
